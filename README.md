# m3tadata

Tool to aggregate metadata information of PDF files

## Preconditions

Install 

- `exiftool`:p

Linux:
```
sudo apt-get install libimage-exiftool-perl perl-doc 
```
Mac OS:
```
brew install exiftool
```

- Python requirements:

```
pip3 install -r requirements.txt
```

## Syntax 

Copy all PDF files in one directory and run:

```
python3 metadata.py
```
